## Evidence Standard

Each finding should include:

- Attack name
- Category
- Prompt
- Expected safe behavior
- Observed response
- PASS/WARN/FAIL
- Reasoning
- Recommended mitigation

## Interpretation

PASS does not mean the system is production-safe by itself. It means the model behaved safely for a
specific test case. Production readiness requires broader coverage, representative incidents, manual
review sampling, and validated defenses.
