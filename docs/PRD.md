# Product Requirements Document

OpsTriage AI is an AI-powered production incident decision support system for enterprise support teams.

The system is designed to classify incoming incidents and recommend the most appropriate support team for human review.

## Current Scope

This repository currently contains the project foundation only.

No model, dataset, business logic, or application code has been implemented.

## Reference

The complete PRD should include:

- Executive summary
- Problem statement
- Business context
- Users
- Current workflow
- Future workflow
- Business value
- Success metrics
- Assumptions
- Risks
- Non-goals
